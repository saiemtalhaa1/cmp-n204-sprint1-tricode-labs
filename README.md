# CMP-N204 Sprint 1 – Tricode Labs

## Project Overview
This project is part of the Software Engineering module (CMP-N204).
Our team is developing a community-based web application aligned with
the theme **“Sharing, exchange and building community.”**

The aim of the application is to help users share resources and services
for mutual benefit rather than financial gain.
